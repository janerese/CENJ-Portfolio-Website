Quiz Maker

Description

It is a learning management system used by The Software Institute. It is an LMS in the point of view of the teacher / professor. There are four navigations: Dashboard, Courses, Teacher Profile, and the main focus is the Quiz Maker.The user can create their own quiz with as many questions as they want and delete any question.

Languages Used

We utilized multiple programming languages to create this website. Our team of skilled developers leveraged their expertise in these languages to build a dynamic and visually appealing site that is easy to use and optimized for performance.
